# Простой ТГ-бот по изучению английского

# [бот](https://web.telegram.org/a/#8694713273) 

## Возможности

При старте (/start) в бд из файла words.json загружаются 30 стандартных слов, регистрируется новый пользователь.
Вы можете добавлять новые слова. Слова вы вводите сами или получаете случайно через http-запрос
Вы можете удалять слова (они остаются в БД, но Вы их больше не увидите)
Общение с ботом происходит в основном через Reply или Inline клавиши

## Команды

/start - для начала
/help - для информации
/count - подсчет кол-ва изучаемыъ слов
/card - начать отгадывать слова

## Файлы

Создание БД в файле database
Основные алгоритмы в main
words -json файл со стандартными словами
Приложена схема БД из DBeaver
